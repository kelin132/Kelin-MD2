- User enters their phone number in the dashboard.
- Backend generates a pairing code using Baileys.
- Pairing code is displayed instantly on the panel.
- User enters the code in WhatsApp.
- Bot connects automatically.
- Session is saved securely.
- Auto reconnect after restart.
